# Spring Boot Schedule App

Proyecto desarrollado para practicar el manejo de fechas y horarios en Spring Boot, utilizando interceptores HTTP y el procesamiento de información temporal dentro de aplicaciones web.

## 🚀 Tecnologías utilizadas

* Java 21
* Spring Boot
* Maven
* Spring MVC
* Jakarta Servlet API

## 📚 Conceptos aplicados

* Interceptores HTTP (`HandlerInterceptor`)
* Manejo de fechas y horas con Java Time API
* Procesamiento de solicitudes HTTP
* Configuración mediante `WebMvcConfigurer`
* Arquitectura MVC
* Inyección de dependencias

## 📂 Estructura del proyecto

```text
src
├── controllers
├── interceptors
├── config
└── SpringbootHorarioApplication
```

## 🎯 Objetivo

Comprender cómo interceptar solicitudes HTTP y trabajar con información temporal dentro de aplicaciones Spring Boot, mostrando horarios, fechas y datos relacionados durante el procesamiento de peticiones.

## ⚙️ Funcionalidades

* Visualización de fecha y hora actual.
* Intercepción de solicitudes HTTP.
* Procesamiento previo y posterior a los controladores.
* Configuración centralizada de interceptores.
* Respuestas dinámicas basadas en información temporal.


## 📖 Aprendizajes

Durante este proyecto se trabajó con:

* Spring MVC.
* Interceptores HTTP.
* Ciclo de vida de una petición.
* Java Time API.
* Configuración de aplicaciones Spring Boot.
* Buenas prácticas en aplicaciones backend.
